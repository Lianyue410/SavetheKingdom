Project title: Save the Kingdom

Module: GAME AND MOBILE APP DEVELOPMENT CMP-6056B

Unity version: Unity 6.0 (6000.0.68f1)

Controls: 

  Move: WASD / Arrow keys
  
  Camera rotate: Right Mouse Button
  
  Zoom: Mouse wheel
  
  Attack: Left Mouse Button
  
  Pause: Pause button (UI)
  
Objective: Collect all mushrooms, then reach the goal before the timer ends.

Win/Lose:

  Win: All mushrooms collected + reach goal
  
  Lose: Time reaches 0 OR player HP reaches 0
